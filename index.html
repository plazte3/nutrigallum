Proceso

¿Cómo se atiende al cliente?

El proyecto maneja varios modelos y, con base en la información del cliente, se identifica cuál es el más adecuado para su necesidad.

1

Se recibe la información

Se revisa el tamaño de la granja, la cantidad de aves y la necesidad del productor.

2

Se define el modelo

Con esos datos se analiza cuál de los modelos disponibles se ajusta mejor al cliente.

3

Se entrega con orientación

El cliente recibe el producto con indicaciones básicas de uso y manejo.
